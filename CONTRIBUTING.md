# Contributing Guidelines for Nishaniyaan (نشانیاں)

We welcome contributions to the Nishaniyaan Islamic Knowledge Archive. To maintain high scholarly integrity, all contributions must adhere to the following rules:

---

## 📜 Principles for Content Submissions

1. **Qur'anic Verses**: Must include clear Arabic text, Surah & Ayah numbers, recognized English translation, and classical Tafsir attribution (e.g., Ibn Kathir, Al-Qurtubi, Al-Tabari).
2. **Hadith Narrations**: Must cite canonical collections (Bukhari, Muslim, Abu Dawud, Tirmidhi, Nasai, Ibn Majah, Musnad Ahmad), Hadith number, Arabic text, English translation, and authentic grading by accepted scholars (e.g., An-Nawawi, Ibn Hajar, Al-Albani).
3. **Verification Badges**: Every entry MUST assign an explicit verification status:
   - 🟢 `AUTHENTIC_QURAN` / `HADITH_SAHIH`
   - 🟡 `HADITH_HASAN`
   - 🟠 `HADITH_DAIF`
   - 🔴 `HADITH_MAWDU`
   - ⚪ `SCHOLARLY_OPINION`
   - 🔵 `HISTORICAL_RECORD`
   - 🟣 `CONTEMPORARY_OBSERVATION`
   - ⚫ `UNVERIFIED_NEWS`

---

## 🛠️ Contribution Workflow

1. Fork the repository `amirhashmilive/nishaniyaan`.
2. Edit JSON files under `assets/data/` (e.g., `signs-minor.json`, `events.json`).
3. Ensure valid JSON formatting.
4. Submit a Pull Request detailing sources and classical references.
