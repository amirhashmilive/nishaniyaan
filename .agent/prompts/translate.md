# Translation Prompt Template

## Task: Translate Content to English

### Instructions
1. Read the input content
2. Identify the source language
3. Translate to English
4. Preserve:
   - Meaning and context
   - Arabic terms with transliteration
   - Technical and religious terminology
5. Verify translation accuracy
6. Provide the translated content

### Translation Rules
- ALL content must be in English
- No non-English text in the final output
- Arabic terms may be kept (e.g., Dajjal, Qiyamah, Sahih)
- Transliteration should be standard and consistent
- Context must be preserved

### Example
| Input | Output |
|-------|--------|
| "دجال سب سے بڑی آزمائش ہے" | "Dajjal is the greatest trial" |
| "यह सबसे बड़ी परीक्षा है" | "This is the greatest trial" |
| "أكبر فتنة هي الدجال" | "The greatest trial is Dajjal" |
