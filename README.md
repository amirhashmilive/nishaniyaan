# Nishaniyaan (نشانیاں) — Comprehensive Islamic Knowledge Archive

**Nishaniyaan** is a zero-dependency static web archive dedicated to cataloging and contextualizing the Signs of the Day of Judgment (*Ashrat as-Sa'ah*). Grounded in the Noble Qur'an, authentic Sunnah, and classical scholarship.

---

## 🌟 Key Features

- **Zero-Dependency Architecture**: Pure HTML5, CSS3 Custom Properties, and ES6+ Vanilla JavaScript. Hosted on GitHub Pages.
- **Glassmorphism Visual System**: Extracted from production design system with dark/light mode switching (`theme.js`).
- **Interactive Chronological Timeline**: Complete era navigation from Creation to Qiyamah and Jannah/Jahannam.
- **Verification Badge System**: Explicit badges for 9 verification levels (🟢 Authentic Qur'an, 🟢 Sahih, 🟡 Hasan, 🟠 Weak, 🔴 Fabricated, ⚪ Scholarly Opinion, 🔵 Historical Record, 🟣 Contemporary Observation, ⚫ Unverified Report).
- **Sub-Millisecond Search Overlay**: Instant search modal (`Ctrl+K` or `/`) indexing all JSON data repositories.
- **Unverified Claims Quarantine**: Dedicated section in Research page safeguarding against unverified viral rumors.
